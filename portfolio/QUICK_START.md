## 🔧 Customization Quick Tips

### 1. Update Your Links
Edit these in `index.html`:
- **Line 38**: GitHub link
- **Line 39**: LinkedIn link  
- **Line 64**: Download CV link
- **Line 509-564**: Contact section URLs

### 2. Change Colors
Edit `:root` variables in `styles.css` (lines 1-40):
```css
--primary: #3b82f6;        /* Main blue */
--secondary: #06b6d4;      /* Cyan accent */
--bg-primary: #0f172a;     /* Dark background */
```

### 3. Modify Section Content
Simply edit the text in `index.html`:
- About section: Lines 92-113
- Skills section: Lines 118-261
- Experience section: Lines 326-410

### 4. Add Your Profile Photo
Add an image in the About section:
```html
<img src="your-photo.jpg" alt="Your Name" class="about__image">
```

## 🐛 Troubleshooting

### Animations not working?
- Check if JavaScript is enabled
- Clear browser cache (Ctrl+Shift+Delete)
- Check console for errors (F12 → Console tab)

### Links not working?
- Update URLs in `index.html`
- Ensure relative paths are correct

### Mobile menu not appearing?
- Check viewport in browser DevTools
- Make sure you're under 1024px width

### Colors looking different?
- Browser might be using system dark mode settings
- Try refreshing the page
- Check color palette in CSS variables
